# SkyView Weather Dashboard

A responsive weather dashboard built using HTML, CSS, and JavaScript. Users can search for a city and view real-time weather information.

## Features

- Search weather by city name
- Displays temperature, humidity, and weather conditions
- Responsive design for mobile and desktop
- Real-time weather data using OpenWeather API

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- OpenWeather API

## Project Structure

Weather-dashboard/
├── index.html
├── style.css
├── app.js
└── README.md

## How to Run

1. Clone the repository
2. Open index.html in your browser
3. Enter a city name and click Search
